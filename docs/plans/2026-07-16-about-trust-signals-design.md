# ABOUT信頼材料再構成 設計

## 目的

トップページのABOUT欄を、会社の基本情報を並べる場所から、顧客・発注担当者が安心して相談・発注できる根拠を示す場所へ変更する。

## 現状の課題

- 創業年、事業数、資本金は会社概要としては必要だが、発注判断を後押しする材料にはなりにくい
- 4つの大きな数字が同じ重要度で並び、組織規模以外の強みが伝わらない
- サイトが保有するISMS、DX認定、ISTQB Goldの信頼材料がフッターまで見えない

## 決定事項

- 数字中心の訴求をやめる
- 社員数106名のみ、対応体制の規模を示す事実として残す
- 残り3枠は「ISO/IEC 27001（ISMS）」「DX認定」「ISTQB® Gold パートナー」に変更する
- 既存の正式ロゴ画像を再利用する
- 2×2グリッドの基本レイアウトは維持し、数字カードではなく信頼材料カードとして再設計する
- 創業年、事業数、資本金は会社概要ページで引き続き確認できるため、トップページからは外す

## 表示内容

1. **106名の組織体制**
   - IT・AIの両事業を支える組織
   - 2026年4月時点
2. **ISO/IEC 27001（ISMS）**
   - 情報セキュリティ管理体制を構築・運用
3. **DX認定事業者**
   - 経営とデジタル変革を一体で推進
4. **ISTQB® Gold パートナー**
   - ソフトウェア品質・テスト分野の専門性

## UI

- 社員数カードは数値を主役にせず、見出し「106名の組織体制」と説明を同じ情報階層で表示する
- 認証カードは上部にロゴ、下部に名称と顧客視点の意味を表示する
- ロゴ領域と本文領域の高さ・余白を揃え、4カードを同一の信頼材料として見せる
- スマートフォンでは2列を維持し、ロゴと日本語名が潰れないサイズに調整する
- ロゴ画像には認証名の代替テキストを付ける

## 将来の差し替え候補

公開可能な実績情報が整理できた後は、DX認定または社員数枠を次の実績指標に差し替える。

- 累計支援社数
- 継続取引率
- 稼働プロジェクト数
- 有資格者数

数値は集計基準、対象期間、公開許可を確認してから掲載する。

## 検証

- トップページから創業年、事業数、資本金のカードがなくなっている
- 社員数と3つの認証が表示される
- 375px、768px、1024px、1440pxで文字切れ・ロゴ潰れ・横スクロールがない
- テスト、lint、本番ビルドを通す
